Font Files

Intel® Clear Sans was chosen for this project because Michael A. Peters, who has some reading disabilities including but not limited to an increased liklihood of epileptic seizures when his eyes get tired, found the font to be exceptionally easy to read compared with the default fonts that many ePub viewers use.

The Intel® Clear Sans fonts are released by Intel® under the terms of the Apache License, Version 2.0.

Original versions of the Clear Sans font files were obtained from https://01.org/clear-sans and are included in the The Complete Work:

  • ClearSans-Regular-wlatin.ttf
  • ClearSans-Italic-wlatin.ttf
  • ClearSans-Bold-wlatin.ttf
  • ClearSans-BoldItalic-wlatin.ttf

The font files were modified to remove glyphs that are not needed outside the context of Western Latin content, thus significantly reducing the size of the files (this is called Font Subsetting and is a common publishing production practice for embedded fonts)

Additionally, Michael A. Peters created four font files to compliment Intel® Clear Sans. Work on those font files is not yet complete, but is complete enough to use here. They also are released under the terms of the Apache License, Version 2.0:

  • ClearSansSCP.ttf
  • ClearSansSCP-Italic.ttf
  • ClearSansSCP-Bold.ttf
  • ClearSansSCP-BoldItalic.ttf

At this time, no other font files are used in The Complete Work.

Accessibility Rights

Libraries, Schools, or other institutions who need to modify this work in order to provide missing accessibility needs for a user with a disability are explicitly allowed to do so without needing to worry about removing the ISBN code or changing the unique-identifier.

It is requested that the accessibility issue be brought to our attention so the issue can be addressed in a future version, or at a minimum so that we can update the schema:accessibilityHazard <meta> node, but no such requirement for contact is made.

Regardless of whether or you not you contact us about the accessibility issue, you have our thanks for helping to reduce an accessibility barrier for someone who wishes to access the content.
