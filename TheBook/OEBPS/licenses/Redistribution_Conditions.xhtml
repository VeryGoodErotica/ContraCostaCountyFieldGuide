Font Files

Intel® Clear Sans was chosen for this project because Michael A. Peters, who has some reading disabilities including but not limited to an increased liklihood of epileptic seizures when his eyes get tired, found Intel® Clear Sans to be exceptionally easy to read on small screen devices compared with the default fonts that many ePub viewers use.

The Intel® Clear Sans fonts are released by Intel® under the terms of the Apache License, Version 2.0.

Original versions of the Clear Sans Font Files were obtained from https://01.org/clear-sans. Modified versions of those files are included in The Complete Work:

  • ClearSans-Regular-wlatin.ttf
  • ClearSans-Italic-wlatin.ttf
  • ClearSans-Bold-wlatin.ttf
  • ClearSans-BoldItalic-wlatin.ttf

The original Font Files were modified to remove glyphs that are not needed outside the context of Western Latin content, thus significantly reducing the size of the files (this is called Font Subsetting and is a common publishing production practice for embedded fonts).

Additionally, Michael A. Peters created four Font Files to compliment Intel® Clear Sans. Work on those Font Files is not yet complete, but is complete enough to use here. They also are released under the terms of the Apache License, Version 2.0:

  • ClearSansSCP.ttf
  • ClearSansSCP-Italic.ttf
  • ClearSansSCP-Bold.ttf
  • ClearSansSCP-BoldItalic.ttf

At this time, no other Font Files are used in The Complete Work.
